# Calculate the interest of a credit card terminal.

## Powered by:

- Python
- Tkinter

## Usage:

1. Compile and run.

```bash
python main.py
```

2. Insert the fixed interest

3. Insert the interest per parcels

4. Insert the number of parcels

5. Click the button!
